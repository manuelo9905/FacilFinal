1 al 6

1. function () {}
Definición:Declara una función, que es un bloque de código reutilizable diseñado para realizar una tarea específica.
Sintaxis: function nombre(parametros) { ...código... } o const nombre = function(parametros) { ...código... }

2. llamarfuncion();
Definición:Ejecuta (o invoca) una función previamente definida.
Sintaxis:nombreDeLaFuncion
(argumentos);

3. setTimeout()
Definición: Ejecuta un bloque de código o una función después de que haya transcurrido un tiempo específico (en milisegundos). Sintaxis: setTimeout(funcion, retrasoEnMilisegundos);

4. window.location.assign()
Definición: Carga un nuevo documento en el navegador, navegando a la URL proporcionada.
Sintaxis: window.location.assign(url);

5. var
Definición: Declara una variable, opcionalmente inicializándola a un valor. Tiene un ámbito de función o global (forma antigua, menos recomendada hoy en día).
Sintaxis: var nombreDeLaVariable = valor;

6. =
Definición: Operador de asignación. Asigna el valor del lado derecho (operando derecho) a la variable del lado izquierdo (operando izquierdo).
Sintaxis:variable = valor;
