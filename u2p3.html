14 al 19

14. document.querySelector('.id').style.opacity
Definición: Establece o devuelve el nivel de opacidad (transparencia, de 0.0 a 1.0) de un elemento seleccionado.
Sintaxis: document.querySelector(selector).style.opacity = valorOpacidad;

15. document.querySelector().classList.remove()
Definición: Busca el primer elemento que coincida con el selector dado y elimina una o más clases CSS de su lista de clases.
Sintaxis: document.querySelector(selector).
classList.remove(clase1, clase2, ...);

16. let variable = valor;
Definición: Declara una variable local con ámbito de bloque, opcionalmente inicializándola a un valor (forma moderna y recomendada).
Sintaxis: let nombreDeLaVariable = valor;

17. if () {}
Definición: Ejecuta un bloque de código si una condición especificada se evalúa como verdadera.
Sintaxis: if (condicion) { ...código a ejecutar... }

18. document.getElementById('Id').src = "string" + variable + "string"
Definición: Busca un elemento por su atributo ID específico y cambia su atributo src (fuente, típicamente de una imagen o iframe) concatenando cadenas de texto y el valor de una variable.
Sintaxis: document.getElementById(id).src = concatenacionDeStringsYVariables;

19. localStorage.setItem()
Definición: Guarda un par clave/valor en el almacenamiento local del navegador, permitiendo que los datos persistan incluso después de cerrar la ventana.
Sintaxis: localStorage.setItem(nombreDeLaClave, valor);
