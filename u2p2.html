Mi espacio

7 al 13

7. new
Definición: Crea una instancia de un objeto definido por el usuario o de un tipo de objeto integrado que tenga un constructor.
Sintaxis: new Constructor(argumentos);

8. Audio()
Definición: Constructor que crea y devuelve un nuevo elemento HTML audio utilizado para manejar la reproducción de sonido.
Sintaxis: new Audio(urlDelArchivoDeAudio);

9. .play();
Definición: Método que intenta iniciar la reproducción del contenido multimedia (audio o vídeo).
Sintaxis: elementoMultimedia.play();

10. document.querySelector().classList.add();
Definición: Busca el primer elemento que coincida con el selector dado y añade una o más clases CSS a su lista de clases.
Sintaxis: document.querySelector(selector).classList.add(clase1, clase2, ...);

11. document.querySelector().style.backgroundColor
Definición: Establece o devuelve el color de fondo de un elemento seleccionado.
Sintaxis: document.querySelector(selector).style.backgroundColor = valorColor;

12. document.querySelector().style.backgroundImage
Definición: Establece o devuelve la imagen de fondo de un elemento seleccionado.
Sintaxis: document.querySelector(selector).style.backgroundImage = "url('ruta_de_la_imagen')";

13. document.querySelector().style.display
Definición: Establece o devuelve el comportamiento de visualización (por ejemplo, block, none, flex) de un elemento seleccionado.
Sintaxis: document.querySelector(selector).style.display = valorDisplay;
